# structural-question

## Purpose

Generate the most structurally consequential question for a theme within a 6–24 month horizon.

## Input

theme: string

## Output Format (strict)

Structural Question:
<one precise question>

Why It Matters:
<2–4 sentences>

Volatility Level:
<low | medium | high>

## Constraints

- Structural, not reactive
- 6–24 month horizon
- No browsing
- No hype
- Deterministic tone
